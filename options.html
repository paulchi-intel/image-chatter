Image Chatter

GNAI image analysis and image generation in a Chrome side panel or popup.

Core Capabilities
  • Load an image from disk through the composer image button
  • Language dropdown (Traditional Chinese, Simplified Chinese, English)
  • Chart and slide prompts for summaries, data extraction, trends, insights, actions, and translation
  • Generate PNG or JPEG images with optional reference images
  • Verify image-capable models from the GNAI documentation
How To Use
  1. Open the extension and enter a GNAI API key.
  2. In Analyze, load an image and use a quick prompt or enter a question.
  3. In Generate, enter a prompt, choose size, quality, and format, then optionally add references.
  4. Choose Verify image analysis/generation models from the model selector to refresh capabilities.
Image Chatter supports GNAI API keys only. ExpertGPT pak_ keys are rejected.
Note: Existing key/model/chat states are stored in chrome.storage.local.
